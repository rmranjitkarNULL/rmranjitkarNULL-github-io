Autonomous Mapping Robot

This project involved developing a frontier-based autonomous mapping robot using ROS and Python. Built on a TurtleBot3 platform, it integrated LiDAR-based SLAM, A* path planning, pure pursuit control, and AMCL localization to autonomously map and explore unknown environments with minimal supervision.

System Architecture

The robot used a modular ROS architecture where gmapping performed SLAM, AMCL handled localization, and custom nodes managed frontier detection and navigation. A* was used for global path planning and pure pursuit for smooth control, creating a robust autonomous exploration loop.

Pure Pursuit

The pure pursuit algorithm allowed smooth trajectory tracking by computing lookahead points and adjusting steering to minimize heading error. This reduced oscillations and improved stability in narrow environments.

Frontier Exploration

The FrontierFinder node identified boundaries between known and unknown areas, clustering frontier cells and ranking them by size and distance. This enabled efficient, goal-oriented exploration.

Monte Carlo Localization

AMCL maintained a probabilistic pose estimate through particle filtering and LIDAR feedback, ensuring accurate localization and robust recovery from drift.

Results

The robot mapped a maze-like environment in 101 seconds, autonomously exploring all frontiers with minimal drift. The A* and pure pursuit combination yielded precise navigation and consistent localization performance.
