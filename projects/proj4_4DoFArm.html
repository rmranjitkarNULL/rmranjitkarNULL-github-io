4-DoF Serial Manipulator: Vision-Guided Pick-and-Place

This project presents a fully integrated vision-guided robotic pick-and-place system built in MATLAB and implemented on a 4-DoF Dynamixel manipulator. The system combines forward and inverse kinematics, differential kinematics, and trajectory generation with computer vision-based perception for autonomous object detection, grasping, and sorting. Using precise intrinsic and extrinsic calibration, the manipulator achieved sub-millimeter localization accuracy and smooth, repeatable motion across the workspace.

System Architecture

The manipulator’s architecture was divided into two modular subsystems: a vision system for real-time object detection and a robot control system for motion execution. A calibrated USB camera captured the workspace and provided object centroids to MATLAB’s ImageProcessor class, which mapped pixels to world coordinates using camera calibration parameters. These target locations were passed to the Robot class, which executed motion commands through blocking joint-space and task-space control routines. This modular framework enabled independent debugging, rapid iteration, and reliable synchronization between perception and control.

Kinematic Modeling & Motion Control

Robot motion was governed by a combination of analytical and numerical inverse kinematics for precision and stability. Forward kinematics were formulated using the standard Denavit–Hartenberg (DH) convention, providing a geometric basis for link transformation. Analytical IK offered efficient closed-form joint solutions, while the Damped Least Squares (DLS) numerical method improved stability near singularities. Smooth, continuous trajectories were produced using cubic and quintic polynomial interpolation, ensuring bounded velocity and acceleration profiles for each pick-and-place cycle.

Computer Vision & Calibration

The vision subsystem employed MATLAB’s Computer Vision Toolbox to implement both intrinsic and extrinsic calibration, mapping 2-D image coordinates into 3-D robot coordinates. A color segmentation pipeline in HSV space isolated colored objects on a checkerboard plane, with centroid detection used to estimate positions in real time. The calibration maintained a mean reprojection error below one pixel, enabling millimeter-level localization accuracy. Through this precise calibration, the manipulator could autonomously detect, classify, and sort multiple colored targets with high repeatability.

Results and Performance

Experimental results demonstrated consistent and precise autonomous manipulation. The system achieved an average localization error of 1.03 mm and a grasp success rate of 94% across 30 trials. The end-effector’s measured trajectories closely matched planned cubic and quintic profiles, confirming the accuracy of the inverse kinematics solver and the Dynamixel servo control. Each complete pick-and-place cycle, including detection, trajectory computation, and grasping, averaged about 7 seconds. These results validated the project’s emphasis on modular system design, calibration accuracy, and smooth trajectory generation as the foundation for reliable autonomous robotic operation.
